# Karunya Institute of Technology and Sciences (KITS) Placement Analytics Portal

This portal is a placement intelligence dashboard for companies recruiting at KITS.

## Phase 2 — Supabase Connection

To connect this portal to Supabase, paste your credentials into the `.env` file at the root:

```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_public_key
```

Then install the dependencies and run the development server:

```bash
npm install
npm run dev
```
