# Placement Analytics Portal

Campus placement research and analytics portal with company profile analysis and skill mapping.

## Phase 2 — Supabase

To connect this portal to your Supabase Postgres database:
1. Paste your `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` into the `.env` file at the project root.
2. Run `npm install && npm run dev` to start the development server.
